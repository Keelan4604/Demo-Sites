# West Alabama Fence — Website Redesign Concept

## What This Is

A complete, production-ready website redesign concept for **West Alabama Fence**, a family-owned fence contractor in Brookwood/Tuscaloosa, Alabama that's been in business since 1997.

Their current website (westalafence.net) is visually dated, lacks conversion optimization, and doesn't reflect the quality of their actual business (4.4 stars, 159 reviews, BBB accredited).

This redesign is a pitch-ready static site built to demonstrate what their web presence could look like — and to close a redesign sale.

## Files

| File | Purpose |
|---|---|
| `index.html` | The redesigned website (single-page, all sections) |
| `styles.css` | All styling — mobile-first, responsive |
| `main.js` | Navigation, smooth scroll, form handling |
| `prospect-analysis.md` | Why this business was chosen, current site problems, and why the redesign improves results |
| `pricing-and-pitch.md` | Pricing tiers ($1,500 / $2,500 / $3,500), sales angle, and offer structure |
| `outreach-email.md` | Cold email draft ready to customize and send |

## Deployment

This is a plain HTML/CSS/JS static site. To deploy:

```bash
# Cloudflare Pages (recommended)
# 1. Push this folder to a GitHub repo
# 2. Connect it to Cloudflare Pages
# 3. Set build output directory to the root
# 4. Deploy

# Or just open index.html locally to preview
```

No build step required. No frameworks. No dependencies.

## Before Pitching

1. **Replace gallery placeholders** with actual photos from their Facebook page or with permission from the owner
2. **Customize the outreach email** with your contact info
3. **Review the pricing** against your local market rates and adjust if needed
4. **Open index.html** in a browser and walk through the full site on both desktop and mobile
